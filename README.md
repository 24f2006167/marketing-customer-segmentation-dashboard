# Marketing Campaign Customer Segmentation Dashboard

## Project Overview

This project focuses on customer segmentation analysis using marketing campaign data.  
The goal is to identify different customer groups based on their income, spending behavior, age, and purchasing patterns.

The project combines:
- Data Analysis
- Customer Segmentation
- Interactive Dashboard Visualization

using Python and Tableau.

---

# Project Objectives

- Analyze customer demographics and spending patterns
- Identify customer segments using clustering
- Create meaningful KPIs
- Build an interactive Tableau dashboard
- Visualize customer behavior for business insights

---

# Dashboard Insights

- Customers are divided into multiple clusters based on income and spending patterns.
- High-income customers generally show higher spending behavior.
- Some customer groups have low spending despite high income.
- The segmentation helps businesses target customers more effectively.
